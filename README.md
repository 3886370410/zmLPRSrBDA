# 项目简介

## 核心功能
- 登录注册
- 考勤打卡
- 考勤统计
- 部门管理
- 员工管理
- 个人设置

## 适用场景
- 企业内部员工考勤管理
- 人力资源管理
- 办公自动化

## 技术特点
- 使用 Django 框架进行后端开发
- 基于 MySQL 数据库进行数据存储
- 前端采用 Vue.js 框架构建用户界面
